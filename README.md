# クロノス邸メイド診断～あなたのお部屋係は誰？～

ブラウザで動くADV風メイド相性診断ゲームです。

## 起動方法

`index.html` をブラウザで開いてください。

## 今回の追加機能

- メイド一覧は初期状態ではシルエット表示です。
- 診断結果で「あなたのお部屋係」として表示されたメイドからプロフィールが解放されます。
- メイド一覧で各メイドのカードをクリックするとプロフィール画面が表示されます。
- 未解放メイドはシルエットと未解放メッセージのみ表示されます。
- メイド一覧の「解放状況リセット」からコレクション状態を初期化できます。
- 冒頭でアウルムが「お部屋係」の朝昼晩の業務内容を説明します。
- Web Audio APIによる上品な室内楽風BGMを追加しました。画面上部の「BGM ON/OFF」で切り替えできます。

## 補足

解放状況はブラウザの localStorage に保存されます。別ブラウザやシークレットウィンドウでは共有されません。
